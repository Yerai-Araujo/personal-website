Este proyecto es una aplicación backend diseñada para resolver un caso de uso real mediante una arquitectura limpia, modular y fácil de extender. El objetivo principal es ofrecer una API robusta que gestione datos de forma eficiente, implementando autenticación segura, validaciones consistentes y patrones de diseño que facilitan el mantenimiento a largo plazo.

Funcionalidades principales

  • Permite a un usuario seguir activos “duros” como Bitcoin y otras criptomonedas.
  • Soporte para seguimiento de oro, plata y otros metales preciosos.
  • Consulta de precios actuales de todos los activos disponibles.
  • Gestión completa de un portfolio personal.
  • Cálculo automático del valor total del portfolio y exposición del usuario.
  • API REST completamente documentada y estructurada en módulos.
  • Consultas a base de datos optimizadas y seguras.
  • Containerización con Docker para despliegue rápido.
  • Clean architecture + DDD.
  • CircuitBreaker y fallback para mayor resiliencia del sistema.
